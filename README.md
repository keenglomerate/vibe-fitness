
# Vibe Fitness 💪

Vibe Fitness is a modern, fully responsive Single Page Application (SPA) built with React. It serves as a comprehensive digital fitness companion, offering users structured workout splits, nutritional advice, recovery techniques, and built-in health metric calculators. 

Built entirely on the front-end for lightning-fast performance, this project demonstrates modern UI/UX principles, dynamic routing, and state management without the need for a backend server.

## ✨ Key Features

* **Real-Time BMI Calculator:** A fully client-side calculator utilizing React `useState` to process weight and height inputs and dynamically render health categories instantly.
* **Dynamic Client-Side Routing:** Seamless page navigation using `react-router-dom` for a smooth, app-like experience without page reloads.
* **Modern UI Design:** Features a dark-mode aesthetic with custom CSS, including frosted glassmorphism effects, dynamic image backgrounds, and overlapping layouts.
* **Comprehensive Content Architecture:** Modular React components broken down by domain (Workout, Nutrition, Progress, Recovery, etc.) for clean, maintainable code.

## 🛠️ Tech Stack

* **Core:** React 18, JavaScript (ES6+)
* **Build Tool:** Vite (for rapid development and optimized builds)
* **Routing:** React Router DOM
* **Styling:** CSS-in-JS / Inline styling for scoped, component-level design

## 🚀 Getting Started

To get a local copy up and running on your machine, follow these simple steps.

### Prerequisites
* Node.js installed (v16.0 or higher recommended)
* Git installed


### Installation

1. **Clone the repo:**
```bash
   git clone https://github.com/keenglomerate/vibe-fitness.git

```

2. **Navigate to the project directory:**
```bash
cd vibe-fitness

```


*(Note: If the code is nested inside a `frontend` folder, run `cd frontend` instead).*
3. **Install NPM packages:**
```bash
npm install

```


4. **Run the development server:**
```bash
npm run dev

```


5. **View the app:**
Open `http://localhost:5173` in your browser to see the application running.
